# Employee Information System

## Description
A simple Java program that stores and displays employee information using Object-Oriented Programming (OOP) concepts.

## Features
- Store employee name
- Store employee salary
- Display employee details
- Uses classes and objects

## Concepts Used
- Classes and Objects
- Data Members
- Member Functions

## Sample Output
```text
Name: Mamoona
Salary: 50000.0
```

## How to Run
```bash
javac Employee.java
java Employee
```
